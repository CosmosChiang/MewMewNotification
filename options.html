About

MewMewNotification

MewMewNotification - Chrome 擴充功能，用於高效接收 Redmine 通知。

Features:

  • Real-time notification checking
  • Desktop notifications with sound alerts
  • Mark notifications as read
  • Clear notification history
  • Direct links to Redmine issues
  • Customizable check intervals
  • Filter by assigned issues
  • Include watched issues
  • Multi-language support (EN/ZH_TW/ZH_CN/JA)
  • Auto-detect issue updates
  • Badge counter for unread notifications
